# 関連研究

この文書は、現時点での接続先を整理するためのメモです。完全な文献レビューではありません。

## 1. 計算創造性

計算創造性は、計算システムが創造的に見える成果物を生成・評価する仕組みを扱う研究領域です。

このリポジトリは、計算創造性のうち、特に「生成された構造をどのように評価するか」という問題に接続します。

## 2. Boden の創造性分類

Margaret Boden は、創造性を大きく次のような方向で整理しています。

- 組合せ型創造性
- 探索型創造性
- 変形型創造性

このリポジトリの再解釈型創造性は、特に「変形型創造性」に近いです。

ただし、このリポジトリは Boden の理論をそのまま実装するものではありません。

## 3. 新規性と有用性

創造性評価では、単に新しいだけでは不十分です。

新規性に加えて、有用性・意味性・適切性が必要です。

このリポジトリでも、単なる誤りやランダムな差分は創造性候補とは呼びません。

## 4. モデル修復

このリポジトリの考え方は、内部不整合を持つモデルを修復し、別の整合的構造を作るという点で、モデル修復や制約充足問題に接続します。

ただし、通常のモデル修復が「正しいモデルへの復元」を重視するのに対し、ここでは「元と異なるが整合的で価値のある代替モデル」を創造性候補として扱います。

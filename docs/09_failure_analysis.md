# 失敗分析: バイアス付き再解釈と集団分布

この文書では、再解釈型創造性モデルの重要な失敗モードを整理します。

## 1. 個人レベル: 自己整合化された誤解

個人の内部ワールドモデルに不整合があるとき、主体はその不整合を減らすように再解釈する場合があります。

この再解釈は、創造性候補になることもあります。

しかし、初期モデルに強いバイアスがある場合、再解釈は誤った前提を保存しながら内部整合性だけを高めることがあります。

この状態を、このリポジトリでは **自己整合化された誤解** と呼びます。

```text
誤った前提
  -> 内部不整合
  -> 再解釈
  -> 内部的には整合しているが、外部基準では誤ったモデル
```

重要なのは、内部整合性だけでは創造性候補として不十分だという点です。

## 2. 集団レベル: 誤りは分布になる

同じ教師モデルを複数の生徒に伝えると、各生徒はそれぞれ異なる内部モデルを作ります。

個人単位では、ある生徒の再解釈は誤った方向に進むことがあります。

しかし、多数の生徒を集団として見ると、それらの誤りは分布になります。

つまり、単一の誤解ではなく、複数の再解釈候補がばらついて現れます。

```text
教師モデル T
  -> 生徒 S1 の再解釈 W1'
  -> 生徒 S2 の再解釈 W2'
  -> 生徒 S3 の再解釈 W3'
  -> ...
```

このとき、再解釈は一つの方向に固定されず、候補集合として広がります。

## 3. 完全ランダムではない

ただし、この分布は完全な一様ランダムではありません。

分布は、次の要因に引っ張られます。

- 教師モデルの構造
- 教材の表現
- 生徒の事前知識
- 生徒のバイアス
- 評価関数
- 文化的・制度的前提

したがって、より正確には次のように言うべきです。

> 個人レベルではバイアス付き再解釈が起きる。集団レベルでは、それらが偏った分布として現れる。

## 4. 複数教師の場合

教師が複数いる場合、誤りが単純に消えるとは限りません。

複数教師が互いに独立した誤差を持つ場合、集団全体では誤りが平均化される可能性があります。

一方で、複数教師が同じバイアスを共有している場合、そのバイアスはむしろ強化されます。

| 条件 | 起きること |
|---|---|
| 教師の誤差が独立 | 誤差が平均化されやすい |
| 教師が同じバイアスを共有 | 誤りが強化される |
| 教師間で評価基準が違う | 再解釈候補が広がる |
| 教師間で矛盾がある | 生徒内部に不整合が入りやすい |

## 5. 生徒間交流とアンサンブル

生徒同士が交流する場合、各生徒の再解釈は独立ではなくなります。

複数の生徒モデルが互いに参照されることで、集団は一種のアンサンブルとして働く可能性があります。

このとき、個々の生徒が誤っていても、集団内の相互比較によって極端な逸脱が抑えられる場合があります。

ただし、集団内に共有バイアスがある場合、交流は誤りを減らすのではなく、誤りを強化する可能性もあります。

したがって、現時点では次の仮説に留めます。

> 生徒集団は、個々の再解釈候補をアンサンブル化することで、極端な逸脱を抑える場合がある。ただし、共有バイアスが強い場合は、集団全体が誤った方向へ収束する可能性がある。

この仮説は重要ですが、初期MVPでは実装しません。

## 6. 創造性との関係

この分析は、再解釈型創造性を弱めるものではありません。

むしろ、次の区別を明確にします。

| 種類 | 内容 |
|---|---|
| 自己整合化された誤解 | 内部的には整合するが、外部妥当性や proxy 上の評価が弱い |
| 再解釈型創造性候補 | 内部整合性・差分・構造的 proxy による評価を満たす |
| 社会的創造性 | さらに他者・共同体・専門家などに評価される場合 |

創造性候補を扱うには、内部整合性だけでなく、構造的 proxy、外部妥当性、評価共同体との関係を分ける必要があります。

## 7. 実装への扱い

この文書の内容は、現時点では将来課題として扱います。

このリポジトリの初期目的は、再解釈型創造性の最小モデルを整理することです。

したがって、現時点では次を行いません。

- 生徒集団シミュレーション
- 生徒間交流ネットワークの実装
- アンサンブル実験のCI追加
- 大規模な集団モデルの導入

必要になった場合は、別リポジトリまたは別フェーズとして切り出します。

## 8. 現時点の結論

再解釈は、創造性にもつながるが、誤った方向にも進みます。

個人レベルでは、これは **自己整合化された誤解** です。

集団レベルでは、それは **偏った再解釈分布** として現れます。

生徒同士が交流する場合、集団はアンサンブルとして機能する可能性があります。ただし、それはこのリポジトリの初期MVPでは扱わず、将来課題に留めます。

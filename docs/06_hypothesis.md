# 検証仮説

このリポジトリでは、創造性全体を証明するのではなく、再解釈型創造性について反証可能な仮説を置きます。

## H: 反証可能仮説

内部不整合を持つ人工ワールドモデル `W` に対して、再解釈探索はランダム修復よりも高い評価スコアを持つ代替モデル `W'` を生成する。

```text
平均スコア(再解釈探索) > 平均スコア(ランダム修復)
```

ここでのスコアは、次の proxy 指標を合成したものです。

- 整合性
- 新規性
- 保存度
- 有用性 proxy

## T: 最小検証

| 項目 | 内容 |
|---|---|
| データ | 3ノードから始める人工ワールドモデル |
| 環境 | Python 標準ライブラリのみ |
| 最小試行 | まず固定デモケース、次に乱数生成ケースへ拡張 |
| 比較対象 | ランダム修復、教師モデル復元、再解釈探索 |
| 停止条件 | テストとデモケースが安定して再現できること |

## D: 合否条件

PASS:

```text
score(reinterpretation) >= score(random_repair)
```

かつ、再解釈後モデルが教師モデルと異なること。

FAIL:

```text
score(reinterpretation) < score(random_repair)
```

または、再解釈後モデルが教師モデルの復元だけになること。

UNCERTAIN:

- 評価関数を少し変えただけで結果が反転する。
- ランダム seed に強く依存する。
- 新規性だけが高く、有用性 proxy が低い。

## C: 破れ方

このモデルは、次の場合に弱い、または破れます。

| 破れ方 | 意味 |
|---|---|
| ランダム修復と差がない | 再解釈探索の意味が弱い |
| 教師モデル復元にしかならない | 創造性候補ではなく正解復元である |
| 差分だけ大きい | ランダムノイズに近い |
| 有用性 proxy が低い | 内部整合した誤解にすぎない |
| ノード数を増やすと崩れる | 小規模例だけの現象である |

## U: 不確かさ

| 不確かさ | 内容 |
|---|---|
| 価値関数 | `utility_proxy` は人間の価値判断ではない |
| グラフ単純化 | 人間の認識構造より単純すぎる |
| 内部不整合定義 | 現時点では肯定・否定の衝突に限定している |
| サンプル数 | 現時点では固定デモケース中心である |
| 社会的受容 | 初期MVPでは扱わない |

## 現時点の結論

このリポジトリの初期段階では、次だけを確認します。

> 小さな人工モデル上で、内部不整合から、元モデルと異なる整合的な代替モデルを生成できるか。

これ以上の主張は、現時点ではしません。

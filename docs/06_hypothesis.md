# 検証仮説

このリポジトリでは、創造性全体を証明するのではなく、再解釈型創造性について反証可能な仮説を置きます。

## H: 反証可能仮説

内部不整合を持つ人工ワールドモデル `W` に対して、再解釈探索は単発ランダム修復よりも高い評価スコアを持つ代替モデル `W'` を生成する。

```text
平均スコア(再解釈探索) > 平均スコア(単発ランダム修復)
```

ただし、これは弱い仮説です。

より強い仮説は、同じ候補数予算を使うランダム探索や局所修復探索よりも、再解釈探索が高い評価スコアを出すことです。

```text
平均スコア(再解釈探索) > 平均スコア(ランダム探索)
平均スコア(再解釈探索) >= 平均スコア(局所修復探索)
```

現時点では、弱い仮説は満たしますが、強い仮説は満たしていません。

ここでのスコアは、次の proxy 指標を合成したものです。

- 新規性
- 保存度
- 有用性 proxy

有用性 proxy は、密度・ノード被覆・弱連結・入出力被覆の暫定的な合成値です。

## T: 最小検証

| 項目 | 内容 |
|---|---|
| データ | 3〜5ノードの人工ワールドモデル |
| 環境 | Python 標準ライブラリのみ |
| 最小試行 | 各ノード数につき10 seed |
| 比較対象 | 単発ランダム修復、ランダム探索、局所修復探索、再解釈探索 |
| 出力 | Markdown / JSON / CSV |
| 説明ログ | 1ケースごとに教師モデル・初期モデル・探索結果・変更辺を出力 |
| 停止条件 | CIで固定デモ、バッチ実験、説明ログ生成、テストが通ること |

## D: 合否条件

弱いPASS:

```text
mean(score(reinterpretation)) > mean(score(single_random_repair))
```

強いPASS:

```text
mean(score(reinterpretation)) > mean(score(random_search))
```

局所修復比較でのPASS:

```text
mean(score(reinterpretation)) >= mean(score(local_repair_search))
```

現時点の状態:

| 条件 | 状態 |
|---|---|
| 単発ランダム修復に勝つ | PASS |
| ランダム探索に勝つ | FAIL 条件あり |
| 局所修復探索以上になる | FAIL 条件あり |

FAIL:

- 再解釈探索が単発ランダム修復にも勝てない。
- 再解釈後モデルが教師モデルの復元だけになる。
- 強いベースラインに一貫して負ける。

UNCERTAIN:

- 評価関数を少し変えただけで結果が反転する。
- ランダム seed に強く依存する。
- 新規性だけが高く、有用性 proxy が低い。
- 説明ログ上、スコアは高いが変更内容が解釈しにくい。

## C: 破れ方

このモデルは、次の場合に弱い、または破れます。

| 破れ方 | 意味 |
|---|---|
| 単発ランダム修復と差がない | 再解釈探索の最低限の意味が弱い |
| ランダム探索に負ける | 候補生成方針が探索として弱い |
| 局所修復探索に負ける | 現在の再解釈探索に局所改善過程が不足している |
| 教師モデル復元にしかならない | 創造性候補ではなく正解復元である |
| 差分だけ大きい | ランダムノイズに近い |
| 有用性 proxy が低い | 内部整合した誤解にすぎない |
| ノード数を増やすと崩れる | 小規模例だけの現象である |
| 説明ログで変更内容が読めない | スコアはあっても説明可能性が弱い |

## U: 不確かさ

| 不確かさ | 内容 |
|---|---|
| 価値関数 | `utility_proxy` は人間の価値判断ではない |
| proxy重み | 密度・被覆・連結性などの重みは暫定値である |
| グラフ単純化 | 人間の認識構造より単純すぎる |
| 内部不整合定義 | 現時点では肯定・否定の衝突に限定している |
| サンプル数 | 現時点では小規模 seed に限定している |
| 探索方針 | 再解釈探索、局所修復探索、ランダム探索の公平性が未完成である |
| 社会的受容 | 初期MVPでは扱わない |

## 現時点の結論

このリポジトリの現段階で確認できるのは、次です。

> 小さな人工モデル上で、内部不整合から、元モデルと異なる整合的な代替モデルを生成し、その変更内容を説明ログで確認できる。

ただし、現在の再解釈探索は、強めのランダム探索や局所修復探索に負ける条件があります。

したがって、現時点の結論は次に留めます。

> 再解釈型創造性の最小モデルと検証枠組みは動作している。だが、探索戦略として十分に強いとはまだ言えない。

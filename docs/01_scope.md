# 射程

このリポジトリは、創造性の全体理論を目指しません。

対象は、**再解釈型創造性** です。これは、主体の内部ワールドモデルに不整合・曖昧さ・衝突が生じ、それを別の整合的な構造へ修復・変形することで生まれる創造性候補を指します。

## 主対象

扱うものは次です。

- 主体の内部ワールドモデル
- 内部不整合
- 不整合を減らす再解釈
- 元モデルとの差分
- 評価基準上の価値
- 人工グラフ上でのシミュレーション

## 背景に留めるもの

次は重要ですが、このリポジトリの主対象ではありません。

- 対話一般
- 共通理解一般
- 行動計画の不一致リスク低減
- 自然言語理解の完全モデル
- 人間心理の完全な記述

これらは、ワールドモデル差異を考える背景としてだけ扱います。

## なぜ範囲を絞るのか

創造性は広い概念です。発明、芸術、科学的発見、熟練者の設計、偶然の発見、ランダム探索からの成果など、多くの経路があります。

このリポジトリで扱うのは、そのうちの一経路だけです。

> 内部不整合を持つモデルが、より整合的な代替モデルへ再解釈され、その代替モデルが有意味である場合。

この射程を明記することで、過剰主張を避けます。

# 用語定義

このリポジトリでは、日常語としての「創造性」と、シミュレーション上の用語を明確に分けます。

## 創造性

このリポジトリでは、創造性全体を定義しません。

対象は、創造性の一部である **再解釈型創造性** です。

## 再解釈型創造性

主体の内部ワールドモデルに生じた不整合が、別の整合的な代替構造へ修復・変形され、その代替構造に評価基準上の価値がある場合に成立する創造性候補です。

これは、創造性の必要十分条件ではありません。

## ワールドモデル

主体が持つ概念・関係・信念の構造です。

初期実装では、自然言語や心理状態を直接扱わず、符号付き有向グラフとして単純化します。

## 内部不整合

主体のモデル内部に、同時には成立しにくい関係が含まれている状態です。

MVPでは、同じ辺に対する肯定と否定の衝突だけを扱います。

```text
A -> B
not (A -> B)
```

## 再解釈

内部不整合を持つモデルから、別の整合的なモデルを作ることです。

ただし、単に教師モデルへ戻すだけの場合は、このリポジトリでいう再解釈型創造性候補としては弱いです。

## 価値 proxy

再解釈後モデルに何らかの意味・有用性があるかを見るための簡易指標です。

これは人間の価値判断そのものではありません。

## 創造性候補

再解釈後モデルが、整合性・差分・価値 proxy を満たす場合の呼び方です。

社会的・専門的に創造的と認められたことまでは意味しません。

## 社会的受容

専門家、利用者、観衆、市場などが価値を認めることです。

初期MVPでは扱いません。背景概念としてのみ位置づけます。

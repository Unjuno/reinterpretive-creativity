# 用語定義

このリポジトリでは、日常語としての「創造性」と、シミュレーション上の用語を明確に分けます。

## 創造性

このリポジトリでは、創造性全体を定義しません。

対象は、創造性の一部として考えられる **再解釈型創造性** の限定モデルです。

## 再解釈型創造性

主体の内部ワールドモデルに生じた不整合が、別の整合的な代替構造へ修復・変形され、その代替構造が明示された構造的 proxy によって評価される場合に成立しうる創造性候補です。

これは、創造性の必要十分条件ではありません。

## ワールドモデル

主体が持つ概念・関係・信念の構造です。

このリポジトリでは、単純な有向グラフとして表します。

## 内部不整合

ワールドモデルの中に、互いに衝突する関係がある状態です。

例:

```text
A -> B
not (A -> B)
```

同じ関係を肯定し、同時に否定しているため、不整合とみなします。

## 再解釈

内部不整合を持つモデルを、別の整合的な構造として捉え直すことです。

ここでは、人間の意味理解そのものではなく、人工グラフ上の構造変換として扱います。

## 構造的 proxy

現在の評価で使う proxy は、人間の価値判断ではありません。

グラフ構造上の distance、preservation、utility_proxy、aggregate score などの測定値です。

## 誤りとの違い

誤りや不整合そのものは創造性ではありません。

誤りは、再解釈のきっかけになることがあります。しかし、創造性候補として扱うには、少なくとも整合性、差分、明示された proxy 上の評価が必要です。

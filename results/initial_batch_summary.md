# 初期バッチ実験サマリ

この結果は、`scripts/run_experiments.py` の固定設定で得られる決定的な初期デモです。

## 条件

| 項目 | 値 |
|---|---:|
| ノード数 | 3 |
| 試行数 | 30 seed |
| conflict_rate | 0.35 |
| extra_positive_rate | 0.25 |
| 比較 | ランダム修復 vs 再解釈探索 |

## 結果

| 指標 | 値 |
|---|---:|
| ランダム修復 平均スコア | 0.1833 |
| 再解釈探索 平均スコア | 0.7394 |
| 平均改善量 | 0.5561 |
| 勝率 | 100% |

## 解釈

この条件では、再解釈探索がランダム修復より高い代理スコアを出しました。

ただし、これは創造性全体を証明するものではありません。次の制約があります。

- 3ノードだけの小規模人工モデルである。
- 価値は `utility_proxy` による代理指標である。
- ランダム修復との比較であり、人間評価ではない。
- 勝率100%は、現時点の単純な探索設定に強く依存している可能性がある。

したがって、この結果は「再解釈型創造性モデルの初期デモが動作している」ことを示すに留めます。
